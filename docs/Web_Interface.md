# Web Interface

The web interface is designed to provide a user-friendly platform for account linking and management. It is responsible for consuming verification tokens generated by any other platform and verifying them.

This is NOT a command based application, but rather a web application that provides a user interface for account linking and management.

## Responsibilities

1. Consuming and verifying verification tokens generated by any other platform.
2. Using login context to determine the user account to associate with the verified token.
3. Account linking and management, including linking and unlinking accounts, and displaying linked accounts.
4. Does not generate verification tokens for any platform.
5. Does not interact with other platforms.
6. Provides a user-friendly platform for account linking and management.
